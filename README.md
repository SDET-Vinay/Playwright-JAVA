
# Playwright TypeScript Web Automation

This repository demonstrates end-to-end (E2E) web automation using https://playwright.dev/ with TypeScript. It includes sample test cases, advanced locator strategies, configuration options, and reporting features to help you build robust automation frameworks.


## Features

>> Cross-browser testing: Chromium, Firefox, WebKit

>>TypeScript support for type safety and modern syntax

>>E2E test cases covering login flows, UI validations, and more

>>Locator strategies: CSS, XPath, and Playwright-specific getBy* methods

>>Parallel execution and project-specific configs
HTML reporting and trace viewer for debugging

>>Playwright Test Runner with advanced options

## Prerequisites

Node.js (v16+ recommended)
npm or yarn 


## CLI Cheat Sheet

npx playwright show-report (View reports from last run)

npx playwright test (Run all the Test cases)

npx playwright test --head (Execute test cases in headed mode)

Refer -  https://playwright.dev/ for more.
